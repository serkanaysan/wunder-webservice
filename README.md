# Wunder Startup Stash Cronjob

> Cronjob with a NodeJS on Linux

## Usage

Install [nodejs](https://nodejs.org/en/)

```
git clone https://github.com/serkanaysan/wunder-cron.git
npm i nodemon -g
```

Install dependencies

```
git clone https://github.com/serkanaysan/wunder-cron.git
cd wunder-cron
npm install
```

To start the root of the project

```
node index.js
```
